• Faculty Feedback System

This repository contains a Faculty Feedback System web application developed as a summer internship project using C# and ASP.NET.
The system enables students to provide structured feedback on faculty performance, and allows administrators to manage and analyze collected feedback.

• About This Project

The Faculty Feedback System is designed to:
Allow students to submit feedback for faculty members , Store responses in a database , Provide administrators with access to review and analyze feedback and
This project was developed to gain practical experience in C#, ASP.NET MVC, database integration, and web form handling as part of a summer internship.

• Key Features

-> Student feedback submission form
-> Faculty listing and selection
-> Database storage of responses
-> Admin panel for feedback review

• Tech Stack
Language: C#
Framework: ASP.NET MVC 
Frontend: HTML, CSS, Js
Database: sql Server Management Studio 21
IDE: Visual Studio 2022
